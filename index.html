Position & Display CSS3

Position relative

Nos permite cambiar la obicacion del elemento relativo a su posicion/ubicacion inicial y nos da acceso a los valores top, bottom, left y right para mover el elemento

Contenedor 1
Contenedor 2
Contenedor 3


        .first_relative, .second_relative, .third_relative {
            height: 100px;
            width: 100px;
        }
        
        .main_container_relative {
            background-color: royalblue;
            height: 500px;
            width: 500px;
        }
        
        .first_relative {
            background-color: brown;
            position: relative;
            top: 400px;
            left: 100px;
        }
        
        .second_relative {
            background-color: aquamarine;
        }
        
        .third_relative {
            background-color: chartreuse;
        }

Position absolute

Ignora el flujo del documento y no le importa la posicion del padre. Con esto se puede posicionar en cualquier lado. Usando un position relative en su contenedor, se puede posicionar dentro de este

Contenedor 1
Contenedor 2
Contenedor 3

            height: 100px;
            width: 100px;
        }
        
        .main_container_absolute {
            background-color: royalblue;
            height: 500px;
            width: 500px;
            position: relative;
        }
        
        .first_absolute {
            background-color: brown;
            position: absolute;

            bottom: 0;

        }

Position fixed

Va a tomar la pantalla y va a posicionar el elemento relativa a ella y va a estar en esa posicion por siempre. Muy util para boton para subir a la parte de arriba de todo de la pagina

        .div_1, .span_1, .span_2, .div_2 {
            font-size: 2em;
        }
        
        .div_1 {
            display: block;
            background-color: red;
        }
        
        .span_1 {
            display: inline;
            background-color: cyan;
        }
        
        .span_2 {
            background-color: purple;
        }
        
        .div_2 {
            background-color: green;
            display: none;
        }
        
        .img {
            display: inline-block;
            width: 300px;
            height: 300px;
        }

display: block; -> es la opcion por defecto que asigna todo el ancho al elemento y hace que el resto de elementos esten por debajo de el

display inline; -> asigna el menor valor posible del ancho permitiendo que los elementos puedan estar en la misma linea

display inline-block; -> lo mismo que inline, es decir que asigna el menor valor posible del ancho, y podemos modificar el ancho y el alto del elemento, cosa que con inline no toma efecto alguno

display none; -> elimina al elemento por completo del documento

flex-wrap: wrap; -> hara que los elementos puedan fluir en varias lineas

flex-shrink: 0; -> deshabilita el encogimiento del elemento cuando cambia el tama√±o de la pantalla

flex-grow: toma el espacio vacio y lo divide por cada columna del flex

flex-basis: le indica desde donde empezar a expandir el elemento flex

order: indica el orden en el flex que va a tener el elemento

flex: VALOR 1, VALOR 2, VALOR 3; -> el VALOR 1 es el flex-grow, el VALOR 2 es el flex-shrink y el VALOR 3 es el flex-basis

grid-template-columns: especifica el numero y el ancho de las columnas en el grid. Se usa fr para determinar la fraccion del grid, es decir, si establecemos 2fr, ocupara 2 fracciones enteras en el grid. Se puede utilizar repeat(VALOR 1, VALOR 2); el VALOR 1 es la cantidad de columnas deseadas y VALOR 2 el ancho de cada una para evitar por ejemplo, escribir: grid-template-columns: 100px 100px 100px; para establecer que queremos tres columnas de 100px cada una

grid-template-rows: especifica el alto para la columna del grid

grid-auto-rows: especifica el alto por defecto para cada columna del grid. Se puede utilizar minmax(VALOR 1, VALOR 2); el VALOR 1 es el alto minimo de la columna y el VALOR 2 es el alto maximo de la columna, el cual si lo establecemos en auto, se va a expandir de acuerdo al contenido

grid-row-gap: espaciado entre cada fila

grid-column-gap: espaciado entre cada columna

grid-gap: espaciado entre cada fila y columna, es decir, grid-row-gap y grid-column-gap juntos en una sola linea

grid-column-start: que columna va a empezar el elemento, siendo el 1 la primera

grid-column-end: en que columna va a terminar el elemento, siendo el -1 la ultima

grid-column: en que columna va a empezar el elemento, seguido de una /, y en cual va a terminar, es decir, grid-column-start y grid-column-end en una sola linea

grid-row-start: en que fila va a empezar el elemento, siendo el 1 la primera

grid-row-end: en que fila va a terminar el elemento

grid-row: en que fila va a empezar el elemento, seguido de una /, y en cual va a terminar, es decir, grid-row-start y grid-row-end en una sola linea
